## Requirements

- Python >= 3.12

&nbsp;

## Dev

1. Run pre-setup script

    ```bash
    python 
    ```

2. Enter virtual environment as explained

3. Run setup script

    ```bash
    pdm run setup 
    ```

4. Run development server using;

    4.1. Django

    ```bash
    python src/manage.py runserver
    ```

    4.2. Tailwind

    ```bash
    python src/manage.py tailwind start
    ```
